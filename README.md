HTML & CSS – Floating Cards:

Questo progetto è un esercizio pratico realizzato per comprendere meglio il funzionamento dei layout con float in CSS e per strutturare una pagina web semplice ma ordinata utilizzando HTML semantico.

Cosa ho imparato:

Strutturare una pagina HTML in sezioni ordinate e leggibili.
Utilizzare tag semantici per migliorare chiarezza e mantenibilità del codice.
Applicare CSS float per creare layout affiancati senza l’uso di flexbox o grid.
Gestire la proprietà clear per evitare problemi di sovrapposizione degli elementi.
Creare piccole card responsive o comunque adattabili grazie alla gestione delle larghezze.
Importare e gestire fogli di stile separati.
Curare gli aspetti base di tipografia, colori e spaziature.

Tecnologie utilizzate:

HTML5 – struttura e contenuto della pagina
CSS3 – styling, layout con float, spaziature
Float layout (vecchia scuola, utile per comprendere basi del posizionamento)
VS Code come editor di sviluppo

HTML & CSS – Floating Cards

This project is a practical exercise created to better understand how float‑based layouts work in CSS and to structure a simple yet organized web page using semantic HTML.
What I learned:

I learned how to structure an HTML page into clear and readable sections, how to use semantic tags to improve clarity and maintainability of the code, how to apply CSS floats to create side‑by‑side layouts without relying on flexbox or grid, and how to manage the clear property to avoid overlapping elements. I also learned how to create small responsive—or at least adaptable—cards by managing widths, how to import and handle separate style sheets, and how to take care of basic aspects such as typography, colors, and spacing.

Technologies used:

HTML5 – structure and content of the page
CSS3 – styling, float layout, spacing
Float layout (old‑school technique, useful for understanding the basics of positioning)
VS Code as the development editor
